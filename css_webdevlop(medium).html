1. What is the difference between relative, absolute, and fixed positioning in CSS?
A They all work the same way but with different performance characteristics
B Relative positions relative to itself, absolute to nearest positioned ancestor, fixed to viewport
C Relative is for text, absolute for images, fixed for backgrounds

✅ Correct! Relative positions relative to itself, absolute positions relative to nearest positioned ancestor (or document if none), and fixed positions relative to the viewport.
❌ Incorrect! The correct answer is B) Relative positions relative to itself, absolute to nearest positioned ancestor, fixed to viewport. These positioning schemes behave differently in the document flow.
